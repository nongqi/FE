# 数据类型

#### JavaScript有哪些数据类型

6 种简单数据类型（也称为原始类型）：Undefine、Null、Boolean、Number、String 和 Symbol。

1 种复杂类型：Object。

#### null 是对象吗

从逻辑上讲，null 值表示一个空对象指针。 所以给 typeof 传一个 null 会返回"object"：

 ```js
 let car = null;
 console.log(typeof car); // "object"
 ```

任何时候只要保存对象，而当时又没有哪个对象可以保存，就要用 null 来填充该变量
